&   => and
|   => or
^   => xor (exclusive or)
<<  => left shift
>>  => right shift

170               => 1010 1010
90                => 0101 1010
170 & 90  => 10   => 0000 1010
170 | 90  => 250  => 1111 1010
170 ^ 90  => 240  => 1111 0000
170 >> 3  => 21   => 0001 0101
170 << 4  => 160  => 1010 0000
